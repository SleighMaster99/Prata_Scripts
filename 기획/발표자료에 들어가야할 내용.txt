- 팀원 소개: 
간단하게 각 팀원이 맡은 역할을 소개해주세요. 
한 명이 열심히 놀고 한 명만 빡세게 했다 하더라도 스스로 선택한 팀원인만큼 점수는 똑같이 나가니까 가능한 솔직하게 적어주면 좋겠습니다.


- 개발한 게임 개요: 
어떠한 형식의 게임을 만들었다는 간단한 소개.
예) 횡스크롤 방식으로 장애물을 뛰어넘고 아이템을 먹는 쿠키런과 같은 게임을 만들었습니다.


- 본인들이 구현한 기능 소개: 
어디서 가져온 것이 아니라 스스로 아이디어를 구상해서 구현한 기능들에 대한 소개. 
코드를 설명하는 것이 아니라 어떻게 동작하는 기능을 구현했는지 설명하세요. 
설명하는데 코드가 있으면 이해하기 쉬울 것같은 경우엔 코드를 언급해도 됩니다. 
가능하면 각 기능들을 동영상으로 캡쳐해서 삽입하면  예) 마우스 왼쪽 버튼을 누르면 점프하고 점프한 상태에서 한 번 더 누르면 이중 점프하는 기능을 구현했습니다. 이 기능을 구현하기 위해 점프 버튼을 누른 횟수를 기록하는 int 변수를 두어 이 값을 통해 이중점프를 해도 되는 지 여부를 확인했습니다.


- 구현하는 동안 어려웠던 점과 해결한 방법: 
말 그대로입니다. 어려웠던 점들에 대해 적고 어떻게 해결했는지
여러분이 고안한 모든 꼼수를 소개해주세요.


- 게임 조작 방법:
여러분이 만든 게임을 플레이하기 위해 어떤 키 조작이 필요한지 설명해주세요.


- 결과 영상:
여러분이 구현한 게임의 플레잉타임이 굉장히 길 수도 있으니까
자랑할만한 핵심부분만 잘 잘라서 동영상으로 캡쳐하여 보여주세요. 학기초에 업로드한 Syllabus에 삽입된 과거 우수 텀프로젝트 영상을 참고하세요.


- 애셋 출처:
애셋스토어나 구글링을 통해 가져온 본인들이 만들지 않은 모든 애셋(맵, 물체, 이미지, 스크립트, 소리 등)에 대해 출처를 밝혀주세요. 출처가 적혀있지 않은 애셋이 있을 경우 감점 처리하겠습니다. 출처에서 확인 시에 이미 기존에 어느 정도 완성된 프로젝트를 가져와서 조금만 다듬은 수준의 결과물이거나(이 경우엔 큰 일 났다고 생각하세요..-_-), 유료 애셋인 경우 상당한 감점이 있을 겁니다.
